# 実装完了の確認 (Walkthrough)

ご要望いただいた「アプリ内の非エンジニア向けユーザーマニュアル」の実装が完了いたしました。
アプリのヘッダーナビゲーションからいつでも参照できる専用のヘルプページをご用意しています。

## 概要

### 追加されたマニュアル画面 (`Manual.tsx`)
専門用語を極力避け、日々の基本的な使い方を以下の4つのセクションに分けて解説しています。
1. **はじめての方へ（初期設定と2段階認証）**
   - 招待メールを受け取ってから、パスワードの設定、およびGoogle Authenticator等を用いた2段階認証（MFA）のQRコードの読み取りとコード入力までの手順をステップ・バイ・ステップで記載しました。
2. **普段のログインに関する操作**
   - メールアドレスとパスワードに加え、6桁のコード入力が必要であることを簡潔に説明しています。
3. **本の借り方・返し方**
   - ダッシュボードを使った「借りる」「返却する」の具体的なボタン操作と、自分の読書状況の見方について記載しています。
4. **管理者向けマニュアル（管理者のみ）**
   - 管理者権限を持つユーザー向けに、ISBNを使った書籍の追加方法や、新しいユーザーの招待・削除に関する基本操作を追加しました。

### ナビゲーションへの組み込み (`App.tsx`, `ProtectedLayout.tsx`)
- いずれの画面を開いているときでも、画面上部のナビゲーションバーに「**使い方（マニュアル）**」リンクが表示されるようにしました。
- ログイン中のすべてのユーザー（一般、管理者とも）がアクセスして確認することができます。
