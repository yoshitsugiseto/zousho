# 実装計画: 画面上部（ヘッダーと検索）のスクロール固定化

## Goal Description
「ダッシュボード（みんなの本棚）」および「書籍マスタ管理」画面にて、ユーザーが大量の書籍リストをスクロールして閲覧している最中でも、ページ上部の「見出し（タイトル・アクションボタン）」や「検索フィルター」が常に画面上部に固定されて表示（Sticky化）されるようにUIを改善します。

---

## User Review Required
> [!NOTE]
> Tailwind CSS の `sticky` クラスおよび `z-index`、さらに背景色（スクロール時の文字透け防止パラメーター）を適用して、レスポンシブな固定ヘッダーを実現します。
> このプランとタスクを確認いただき、実装に進めてもよろしいでしょうか？

---

## Proposed Changes

### 1. ダッシュボードの固定ヘッダー化
#### [MODIFY] `frontend/src/pages/Dashboard.tsx`
- コンポーネントのルートにある `div` 内の上部（見出し「みんなの本棚」と「検索入力欄」）をラップする新しい `div` を定義します。
- このラッパーに `sticky top-0 z-10 bg-gray-50 pb-4 pt-2 -mx-4 px-4 sm:-mx-6 sm:px-6 lg:-mx-8 lg:px-8` などのクラスを付与し、画面の最上部に吸着させます。背景色（`bg-gray-50`）を敷くことで、スクロールして裏側を通る書籍カードが透けないようにします。

### 2. 書籍マスタ管理の固定ヘッダー化
#### [MODIFY] `frontend/src/pages/AdminBooks.tsx`
- 同じく、コンポーネントのルート直下にある上部（見出し「書籍マスタ管理」、「新しい書籍を登録」ボタン、および「検索入力欄」）をラップする `div` を定義します。
- Dashboard同様に、ラッパーへ `sticky top-0 z-10 bg-gray-50 pb-4 pt-2 -mx-4 px-4 sm:-mx-6 sm:px-6 lg:-mx-8 lg:px-8` 等のクラスを付与し、画面上部に固定します。

---

## Verification Plan

### Manual Verification
1. **ダッシュボード画面のテスト**: 書籍がたくさん並んでいる状態で下にスクロールし、検索バーおよび「みんなの本棚」見出しが画面上部に固定され、裏側のカードが透けないことを確認する。
2. **書籍マスタ管理画面のテスト**: 下にスクロールした際に、検索バーや「新規登録」ボタンが画面上部に固定され、一覧部分（テーブル・リスト）だけがスクロールされることを確認する。
3. **レスポンシブテスト**: スマホサイズ（幅を狭めた状態）でも、左右のパディングやレイアウト崩れがなく綺麗に固定されるか確認する。
